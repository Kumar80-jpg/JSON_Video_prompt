# 🎬 AI Video Prompt Generator (Tamil)

This project contains structured JSON prompts for generating AI videos using tools like Google Flow.

## 📁 Files

- `main.py` → Generates prompt JSON files
- `prompts/part1.json` → First half video (0–8 sec)
- `prompts/part2.json` → Second half video (8–15 sec)

## 🚀 How to Use

1. Run the script:


2. Upload generated JSON files into your AI video tool.

3. Generate:
- Part 1 video (8 sec)
- Part 2 video (7 sec)

4. Merge using:
- CapCut / Premiere Pro

## ✅ Features

- Tamil voice optimized
- No subtitles (avoids spelling issues)
- Perfect timing split
- 4K ready prompts

## 🔥 Output

Final combined video:
- 15 seconds
- Clean audio
- No sync issues
